<h2> Outline of computer science</h2>
<p><strong><a title="Computer science" href="https://en.wikipedia.org/wiki/Computer_science">Computer science</a></strong>&nbsp;(also called&nbsp;<strong>computing science</strong>) is the study of the theoretical foundations of&nbsp;<a title="Information" href="https://en.wikipedia.org/wiki/Information">information</a>&nbsp;and&nbsp;<a title="Computation" href="https://en.wikipedia.org/wiki/Computation">computation</a>&nbsp;and their implementation and application in&nbsp;<a title="Computer" href="https://en.wikipedia.org/wiki/Computer">computer</a>&nbsp;systems. One well known subject classification system for computer science is the&nbsp;<a title="ACM Computing Classification System" href="https://en.wikipedia.org/wiki/ACM_Computing_Classification_System">ACM Computing Classification System</a>&nbsp;devised by the&nbsp;<a title="Association for Computing Machinery" href="https://en.wikipedia.org/wiki/Association_for_Computing_Machinery">Association for Computing Machinery</a>.</p>

<h2><span id="Subfields" class="mw-headline">Subfields</span></h2>
<h3><span id="Mathematical_foundations" class="mw-headline">Mathematical foundations</span></h3>
<ul>
<li><a title="Coding theory" href="https://en.wikipedia.org/wiki/Coding_theory">Coding theory</a>&nbsp;&ndash; Useful in networking and other areas where computers communicate with each other.</li>
<li><a title="Game theory" href="https://en.wikipedia.org/wiki/Game_theory">Game theory</a>&nbsp;&ndash; Useful in artificial intelligence and&nbsp;<a title="Cybernetics" href="https://en.wikipedia.org/wiki/Cybernetics">cybernetics</a>.</li>
<li><a class="mw-redirect" title="Discrete Mathematics" href="https://en.wikipedia.org/wiki/Discrete_Mathematics">Discrete Mathematics</a></li>
<li><a title="Graph theory" href="https://en.wikipedia.org/wiki/Graph_theory">Graph theory</a>&nbsp;&ndash; Foundations for data structures and searching algorithms.</li>
<li><a title="Mathematical logic" href="https://en.wikipedia.org/wiki/Mathematical_logic">Mathematical logic</a>&nbsp;&ndash;&nbsp;<a class="mw-redirect" title="Boolean logic" href="https://en.wikipedia.org/wiki/Boolean_logic">Boolean logic</a>&nbsp;and other ways of&nbsp;<a title="Mathematical model" href="https://en.wikipedia.org/wiki/Mathematical_model">modeling</a>&nbsp;logical queries; the uses and limitations of formal proof methods</li>
<li><a title="Number theory" href="https://en.wikipedia.org/wiki/Number_theory">Number theory</a>&nbsp;&ndash; Theory of the&nbsp;<a title="Integer" href="https://en.wikipedia.org/wiki/Integer">integers</a>. Used in&nbsp;<a title="Cryptography" href="https://en.wikipedia.org/wiki/Cryptography">cryptography</a>&nbsp;as well as a test domain in&nbsp;<a title="Artificial intelligence" href="https://en.wikipedia.org/wiki/Artificial_intelligence">artificial intelligence</a>.</li>
</ul>
<h3><span id="Algorithms_and_data_structures" class="mw-headline">Algorithms and data structures</span></h3>
<ul>
<li><a class="mw-redirect" title="Algorithms" href="https://en.wikipedia.org/wiki/Algorithms">Algorithms</a>&nbsp;&ndash; Sequential and parallel computational procedures for solving a wide range of problems.</li>
<li><a title="Data structure" href="https://en.wikipedia.org/wiki/Data_structure">Data structures</a>&nbsp;&ndash; The organization and manipulation of data.</li>
</ul>
<h3><span id="Artificial_intelligence" class="mw-headline">Artificial intelligence</span></h3>
<p><a title="Outline of artificial intelligence" href="https://en.wikipedia.org/wiki/Outline_of_artificial_intelligence">Outline of artificial intelligence</a></p>
<ul>
<li><a title="Artificial intelligence" href="https://en.wikipedia.org/wiki/Artificial_intelligence">Artificial intelligence</a>&nbsp;&ndash; The implementation and study of systems that exhibit an autonomous intelligence or behavior of their own.</li>
<li><a title="Automated reasoning" href="https://en.wikipedia.org/wiki/Automated_reasoning">Automated reasoning</a>&nbsp;&ndash; Solving engines, such as used in&nbsp;<a title="Prolog" href="https://en.wikipedia.org/wiki/Prolog">Prolog</a>, which produce steps to a result given a query on a fact and rule database, and&nbsp;<a title="Automated theorem proving" href="https://en.wikipedia.org/wiki/Automated_theorem_proving">automated theorem provers</a>&nbsp;that aim to prove&nbsp;<a class="mw-redirect" title="Mathematical theorem" href="https://en.wikipedia.org/wiki/Mathematical_theorem">mathematical theorems</a>&nbsp;with some assistance from a programmer.</li>
<li><a title="Computer vision" href="https://en.wikipedia.org/wiki/Computer_vision">Computer vision</a>&nbsp;&ndash; Algorithms for identifying three-dimensional objects from a two-dimensional picture.</li>
<li><a class="new" title="Soft computing (page does not exist)" href="https://en.wikipedia.org/w/index.php?title=Soft_computing&amp;action=edit&amp;redlink=1">Soft computing</a>, the use of inexact solutions for otherwise extremely difficult problems:
<ul>
<li><a title="Machine learning" href="https://en.wikipedia.org/wiki/Machine_learning">Machine learning</a>&nbsp;- Automated creation of a set of rules and axioms based on input.</li>
<li><a class="mw-redirect" title="Evolutionary computing" href="https://en.wikipedia.org/wiki/Evolutionary_computing">Evolutionary computing</a>&nbsp;- Biologically inspired algorithms.</li>
</ul>
</li>
<li><a title="Natural language processing" href="https://en.wikipedia.org/wiki/Natural_language_processing">Natural language processing</a>&nbsp;- Building systems and algorithms that analyze, understand, and generate natural (human) languages.</li>
<li><a title="Robotics" href="https://en.wikipedia.org/wiki/Robotics">Robotics</a>&nbsp;&ndash; Algorithms for controlling the behaviour of robots.</li>
</ul>
<h3><span id="Communication_and_security" class="mw-headline">Communication and security</span></h3>
<ul>
<li><a class="mw-redirect" title="Computer networking" href="https://en.wikipedia.org/wiki/Computer_networking">Networking</a>&nbsp;&ndash; Algorithms and protocols for reliably communicating data across different shared or dedicated media, often including&nbsp;<a class="mw-redirect" title="Error correction" href="https://en.wikipedia.org/wiki/Error_correction">error correction</a>.</li>
<li><a title="Computer security" href="https://en.wikipedia.org/wiki/Computer_security">Computer security</a>&nbsp;&ndash; Practical aspects of securing computer systems and computer networks.</li>
<li><a title="Cryptography" href="https://en.wikipedia.org/wiki/Cryptography">Cryptography</a>&nbsp;&ndash; Applies results from complexity, probability, algebra and number theory to invent and&nbsp;<a class="mw-redirect" title="Cryptoanalysis" href="https://en.wikipedia.org/wiki/Cryptoanalysis">break codes</a>, and analyze the security of&nbsp;<a class="mw-redirect" title="Cryptographic protocols" href="https://en.wikipedia.org/wiki/Cryptographic_protocols">cryptographic protocols</a>.</li>
</ul>
<h3><span id="Computer_architecture" class="mw-headline">Computer architecture</span></h3>
<ul>
<li><a title="Computer architecture" href="https://en.wikipedia.org/wiki/Computer_architecture">Computer architecture</a>&nbsp;&ndash; The design, organization, optimization and verification of a computer system, mostly about&nbsp;<a class="mw-redirect" title="CPU" href="https://en.wikipedia.org/wiki/CPU">CPUs</a>&nbsp;and&nbsp;<a class="mw-redirect" title="Memory (computers)" href="https://en.wikipedia.org/wiki/Memory_(computers)">Memory</a>&nbsp;subsystem (and the bus connecting them).</li>
<li><a title="Operating system" href="https://en.wikipedia.org/wiki/Operating_system">Operating systems</a>&nbsp;&ndash; Systems for managing computer programs and providing the basis of a usable system.</li>
</ul>
<h3><span id="Computer_graphics" class="mw-headline">Computer graphics</span></h3>
<ul>
<li><a title="Computer graphics" href="https://en.wikipedia.org/wiki/Computer_graphics">Computer graphics</a>&nbsp;&ndash; Algorithms both for generating visual images synthetically, and for integrating or altering visual and spatial information sampled from the real world.</li>
<li><a class="mw-redirect" title="Image processing" href="https://en.wikipedia.org/wiki/Image_processing">Image processing</a>&nbsp;&ndash; Determining information from an image through computation.</li>
<li><a title="Information visualization" href="https://en.wikipedia.org/wiki/Information_visualization">Information visualization</a>&nbsp;&ndash; Methods for representing and displaying abstract data to facilitate human interaction for exploration and understanding.</li>
</ul>
<h3><span id="Concurrent.2C_parallel.2C_and_distributed_systems"></span><span id="Concurrent,_parallel,_and_distributed_systems" class="mw-headline">Concurrent, parallel, and distributed systems</span></h3>
<ul>
<li><a title="Parallel computing" href="https://en.wikipedia.org/wiki/Parallel_computing">Parallel computing</a>&nbsp;- The theory and practice of simultaneous computation; data safety in any multitasking or multithreaded environment.</li>
<li><a title="Concurrency (computer science)" href="https://en.wikipedia.org/wiki/Concurrency_(computer_science)">Concurrency (computer science)</a>&nbsp;&ndash; Computing using multiple concurrent threads of execution, devising algorithms for solving problems on multiple processors to achieve maximal speed-up compared to sequential execution.</li>
<li><a title="Distributed computing" href="https://en.wikipedia.org/wiki/Distributed_computing">Distributed computing</a>&nbsp;&ndash; Computing using multiple computing devices over a network to accomplish a common objective or task and thereby reducing the latency involved in single processor contributions for any task.</li>
</ul>
<h3><span id="Databases" class="mw-headline">Databases</span></h3>
<p><a title="Outline of databases" href="https://en.wikipedia.org/wiki/Outline_of_databases">Outline of databases</a></p>
<ul>
<li><a class="mw-redirect" title="Relational databases" href="https://en.wikipedia.org/wiki/Relational_databases">Relational databases</a>&nbsp;&ndash; the&nbsp;<a title="Set theory" href="https://en.wikipedia.org/wiki/Set_theory">set theoretic</a>&nbsp;and algorithmic foundation of databases.</li>
<li><a class="mw-redirect" title="Structured Storage" href="https://en.wikipedia.org/wiki/Structured_Storage">Structured Storage</a>&nbsp;- non-relational databases such as&nbsp;<a title="NoSQL" href="https://en.wikipedia.org/wiki/NoSQL">NoSQL</a>&nbsp;databases.</li>
<li><a title="Data mining" href="https://en.wikipedia.org/wiki/Data_mining">Data mining</a>&nbsp;&ndash; Study of algorithms for searching and processing information in documents and databases; closely related to&nbsp;<a title="Information retrieval" href="https://en.wikipedia.org/wiki/Information_retrieval">information retrieval</a>.</li>
</ul>
<h3><span id="Programming_languages_and_compilers" class="mw-headline">Programming languages and compilers</span></h3>
<ul>
<li><a class="mw-redirect" title="Compiler theory" href="https://en.wikipedia.org/wiki/Compiler_theory">Compiler theory</a>&nbsp;&ndash; Theory of&nbsp;<a title="Compiler" href="https://en.wikipedia.org/wiki/Compiler">compiler</a>&nbsp;design, based on&nbsp;<a title="Automata theory" href="https://en.wikipedia.org/wiki/Automata_theory">Automata theory</a>.</li>
<li><a title="Programming language" href="https://en.wikipedia.org/wiki/Programming_language">Programming language pragmatics</a>&nbsp;&ndash; Taxonomy of programming languages, their strength and weaknesses. Various&nbsp;<a title="Programming paradigm" href="https://en.wikipedia.org/wiki/Programming_paradigm">programming paradigms</a>, such as&nbsp;<a title="Object-oriented programming" href="https://en.wikipedia.org/wiki/Object-oriented_programming">object-oriented programming</a>.</li>
<li><a title="Programming language theory" href="https://en.wikipedia.org/wiki/Programming_language_theory">Programming language theory</a></li>
<li><a class="mw-redirect" title="Formal semantics of programming languages" href="https://en.wikipedia.org/wiki/Formal_semantics_of_programming_languages">Formal semantics</a>&nbsp;&ndash; rigorous mathematical study of the meaning of programs.</li>
<li><a title="Type theory" href="https://en.wikipedia.org/wiki/Type_theory">Type theory</a>&nbsp;&ndash; Formal analysis of the types of data, and the use of these types to understand properties of programs &mdash; especially program safety.</li>
</ul>
<h3><span id="Scientific_computing" class="mw-headline">Scientific computing</span></h3>
<ul>
<li><a title="Computational science" href="https://en.wikipedia.org/wiki/Computational_science">Computational science</a>&nbsp;&ndash; constructing&nbsp;<a class="mw-redirect" title="Scientific modeling" href="https://en.wikipedia.org/wiki/Scientific_modeling">mathematical models</a>&nbsp;and&nbsp;<a title="Numerical analysis" href="https://en.wikipedia.org/wiki/Numerical_analysis">quantitative analysis</a>&nbsp;techniques and using computers to analyze and solve&nbsp;<a class="mw-redirect" title="Scientific" href="https://en.wikipedia.org/wiki/Scientific">scientific</a>&nbsp;problems.</li>
<li><a title="Numerical analysis" href="https://en.wikipedia.org/wiki/Numerical_analysis">Numerical analysis</a>&nbsp;&ndash; Approximate numerical solution of mathematical problems such as&nbsp;<a class="mw-redirect" title="Root-finding algorithm" href="https://en.wikipedia.org/wiki/Root-finding_algorithm">root-finding</a>,&nbsp;<a title="Numerical integration" href="https://en.wikipedia.org/wiki/Numerical_integration">integration</a>, the&nbsp;<a class="mw-redirect" title="Numerical ordinary differential equations" href="https://en.wikipedia.org/wiki/Numerical_ordinary_differential_equations">solution of ordinary differential equations</a>; the approximation of&nbsp;<a title="Special functions" href="https://en.wikipedia.org/wiki/Special_functions">special functions</a>.</li>
<li><a class="mw-redirect" title="Symbolic computation" href="https://en.wikipedia.org/wiki/Symbolic_computation">Symbolic computation</a>&nbsp;&ndash; Manipulation and solution of expressions in symbolic form, also known as&nbsp;<a title="Computer algebra" href="https://en.wikipedia.org/wiki/Computer_algebra">Computer algebra</a>.</li>
<li><a title="Computational physics" href="https://en.wikipedia.org/wiki/Computational_physics">Computational physics</a>&nbsp;&ndash; Numerical simulations of large non-analytic systems</li>
<li><a title="Computational chemistry" href="https://en.wikipedia.org/wiki/Computational_chemistry">Computational chemistry</a>&nbsp;&ndash; Computational modelling of theoretical chemistry in order to determine chemical structures and properties</li>
<li><a title="Bioinformatics" href="https://en.wikipedia.org/wiki/Bioinformatics">Bioinformatics</a>&nbsp;and&nbsp;<a title="Computational biology" href="https://en.wikipedia.org/wiki/Computational_biology">Computational biology</a>&nbsp;&ndash; The use of computer science to maintain, analyse, store&nbsp;<a class="mw-redirect" title="Biological data" href="https://en.wikipedia.org/wiki/Biological_data">biological data</a>&nbsp;and to assist in solving biological problems such as&nbsp;<a title="Protein folding" href="https://en.wikipedia.org/wiki/Protein_folding">Protein folding</a>, function prediction and&nbsp;<a class="mw-redirect" title="Phylogeny" href="https://en.wikipedia.org/wiki/Phylogeny">Phylogeny</a>.</li>
<li><a title="Computational neuroscience" href="https://en.wikipedia.org/wiki/Computational_neuroscience">Computational neuroscience</a>&nbsp;&ndash; Computational modelling of&nbsp;<a title="Neurophysiology" href="https://en.wikipedia.org/wiki/Neurophysiology">neurophysiology</a>.</li>
</ul>
<h3><span id="Software_engineering" class="mw-headline">Software engineering</span></h3>
<p><a title="Outline of software engineering" href="https://en.wikipedia.org/wiki/Outline_of_software_engineering">Outline of software engineering</a></p>
<ul>
<li><a title="Formal methods" href="https://en.wikipedia.org/wiki/Formal_methods">Formal methods</a>&nbsp;&ndash; Mathematical approaches for describing and reasoning about software design.</li>
<li><a title="Software engineering" href="https://en.wikipedia.org/wiki/Software_engineering">Software engineering</a>&nbsp;&ndash; The principles and practice of designing, developing, and testing programs, as well as proper engineering practices.</li>
<li><a class="mw-redirect" title="Algorithm design" href="https://en.wikipedia.org/wiki/Algorithm_design">Algorithm design</a>&nbsp;&ndash; Using ideas from algorithm theory to creatively design solutions to real tasks.</li>
<li><a title="Computer programming" href="https://en.wikipedia.org/wiki/Computer_programming">Computer programming</a>&nbsp;&ndash; The practice of using a programming language to implement algorithms.</li>
<li><a title="Human&ndash;computer interaction" href="https://en.wikipedia.org/wiki/Human%E2%80%93computer_interaction">Human&ndash;computer interaction</a>&nbsp;&ndash; The study and design of computer interfaces that people use.</li>
